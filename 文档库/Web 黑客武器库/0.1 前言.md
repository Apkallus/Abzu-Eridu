# Web Hacking Arsenal（网络黑客武器库）

在数字时代，Web 应用构成了我们互联互通的核心，而《Web Hacking Arsenal：现代 Web 渗透测试实用指南》一书，则成为掌握 Web 应用渗透测试艺术与科学的必备指南。本书由该领域专家撰写，超越了传统方法，融合了现实世界渗透测试的洞见与综合研究，旨在弥合网络安全领域的关键知识鸿沟，为读者提供理论理解与实践技能。本书的独特之处在于，它聚焦于渗透测试中遇到的实际难题，超越了模拟场景，提供了真实场景的深入剖析。

《Web Hacking Arsenal》的核心在于其适应 Web 安全威胁不断演进本质的能力。它不仅让读者为今日的挑战做好准备，也为未来未预见的复杂性奠定基础。这种前瞻性的方法确保了本书的长久价值，使读者能够在不断变化的网络安全格局中保持领先。

## 主要特点

- 基于真实场景和丰富实战经验，深入探索 Web 应用渗透测试。
- 全面覆盖当代及新兴 Web 安全威胁，并提供可适应未来挑战的策略。
- 理论与实践完美融合，包括来自实际渗透测试的案例研究和实践示例。
- 提供在竞争激烈的漏洞赏金计划中获得优势的战略洞察。
- 详细分析最新的漏洞测试技术，使其有别于该领域的现有文献。

本书不仅仅是指导手册，更是一套基础工具，赋能处于任何学习阶段的读者。无论你是初学者还是希望提升现有技能，本书都能为你打下坚实的基础，并在此基础上不断精进，最终让你不仅拥有丰富的知识，还具备进阶的技能。对于任何希望在不断演变的 Web 应用安全领域留下印记的人来说，这都是一本必读之作。

# Foreword

那么，恭喜你，亲爱的读者！你手中拿着的，正是一本关于我们这个时代奥秘、秘传知识与强大咒语的令人愉快的巨著。我这么说，是发自内心最真诚的。Rafay Baloch 不亚于一位超凡脱俗的魔法师，拥有远超凡人的力量，而我认识这位疯狂的术士已有近十年之久。

早在 2014 年，就智能手机市场份额而言，Android 和苹果 iPhone 已经远远甩开了 RiM 黑莓和 Windows Phone。Android 移动操作系统领先苹果移动操作系统（当时称为 iOS）约十个百分点，当时大家普遍认为这是因为 Android 手机制造成本更低，购买价格也更便宜。

当然，那时智能手机正在席卷全球。即便在当时，你的手机也正在成为未来学家和超人类主义者所说的“外皮层”（exocortex）——一种存在于你大脑之外，却以超乎普通人的记忆和计算能力来辅助大脑的设备。我们确实在进化，使用这些设备来存储和检索我们主观生活中所有最重要、最私密的细节。可以说，移动设备的安全性极其重要，这一点毫不夸张。

再说回 2014 年：那时，我在 Rapid7 工作，负责运营一个社区漏洞披露项目，主要面向那些为公开已知漏洞编写 Metasploit 模块的人。Metasploit 中几乎所有的漏洞利用模块，针对的都是被称为“n-day”的漏洞，即已经公开且软件厂商名义上已修复的漏洞。这类漏洞利用与“零日”（zero-day）漏洞不同，后者是全新的、尚未有补丁的新漏洞。这些 Metasploit 模块可用于测试防御措施、验证补丁，或是发现那些因种种原因尚未打补丁而被忽视的系统。但偶尔也会有研究人员发现零日漏洞，并寻求我的帮助，以便将这些发现披露给软件厂商，从而促成补丁的发布。

于是，Rafay 出现了，他出现在我的收件箱里。我最初以为，这又是一个影响某些低端 Android 智能手机的、相当普通的漏洞披露。挺酷的，但说到底，相当普通。漏洞时有发生，而 Android 作为一个复杂的软件项目，又运行在各种各样的平台上，出现一些有趣的漏洞不足为奇，尤其是在旧版本中。此外，由于谷歌是 Android 背后的主要力量，我们都以为他们对 Android 漏洞的响应速度，会和对他们的旗舰产品 Google Chrome 一样快。我们期待一个简单的“发现-报告-修补”周期。

然而，接下来发生的事情让我大吃一惊。事实证明，Rafay 在 Android 自带的 WebView 浏览器（在 Android 统一使用 Chromium 之前）中发现了一些相当严重的漏洞。更重要的是，这些漏洞似乎难以修复。事实上，它们难修到让谷歌的安全决策者临时制定了一项政策（可能当场决定的）：他们不会修复代号为“Jelly Bean”（即 Android 4.3）的“不受支持”的 Android 操作系统版本。这项政策从未在任何地方正式公布过，而且还忽略了一个事实：当时，全球绝大多数 Android 手机（约占 70% 左右）都在运行 4.3 或更早的版本。按照实际数量计算，这涉及近十亿台设备。

再说一次，这些是人们在日常生活中几乎当作“第二大脑”来依赖的设备。一旦你失去了对它的控制，就意味着你失去了对整个人生细节的控制。在与 Rafay 合作的过程中，我逐渐清楚地认识到：第一，我们（作为人类）绝对需要更加关注核心的 Android 漏洞，并由此扩展到移动环境下整个 Web 客户端和服务器的漏洞世界；第二，Rafay 是世界上极少数有能力且愿意去发现、阐述并公开这类软件漏洞的人之一。

全球的政府机构和企业实体都雇佣像 Rafay 这样的人，来探测我们所依赖的技术（如移动设备、Web 浏览器和 Web 应用程序）中的软件弱点和漏洞。我敢保证，他们此刻正掌握着秘密的漏洞利用程序。但与 Rafay 不同的是，那些人通常发誓保密，天知道他们在用这些私密的、秘密的知识做什么。幸运的是，世界上还有少数几个拥有 Rafay 这种特殊技能的人，愿意公开发表他们的发现。我们依赖公共安全研究人员的努力（尤其是 Rafay Baloch），来增进我们对这些技术实际运作方式和实际实现方式的集体理解。然后我们可以利用这些知识，使它们更能抵御间谍和罪犯的侵害。这就是核心黑客文化的体现：信息渴望自由，而危险的信息往往会因公开披露而被削弱其危害性。

多年来，我很高兴能认识 Rafay 并与他共事，帮助人们在他们的计算生活中（实际上就是他们的现实生活）保持安全。我希望，如果你正在阅读这本书，你能从 Rafay 毕生的工作中学到一两点，并努力以协调一致的、最终是公开的方式，将你的下一个漏洞披露给技术公司或开源项目。分享我们的所学，确实是让我们在这个高度互联和分布式的世界中更好地捍卫我们的隐私、安全和人身安全的最佳途径，或许也是唯一的途径。

——Tod Beardsley，非道德黑客 / Huge Success LLC，PacketFu Security

# 前言

今天，我们与网络世界的联系比历史上任何时期都更加密不可分。我们的生活围绕着技术和众多的 Web 应用程序展开。绝大多数人并不了解它们的工作原理，也并不真正关心。他们只想知道这些技术是否安全可靠。然而，正因这一趋势日益增长，对强大 Web 安全的需求如今比以往任何时候都更为关键。

我在网络安全领域，尤其是 Web 应用程序渗透测试方面的旅程，始终由一种需求、对知识的渴望，以及致力于提升保护我们个人和职业生活的数字防御的承诺所驱动。

本书凝聚了我在真实环境渗透测试和漏洞赏金计划中积累的广泛研究与第一手经验。它试图填补一些关键的知识空白，并希望能成为一本来自该领域实际经验者的实用指南。

本书一个显著的特点在于，它关注 Web 安全威胁的动态特性，并认识到数字威胁、策略和技术在不断演进。书中不仅介绍了当前有效的技术，还注重其适应未来挑战的能力。这保证了本书的长期相关性，并将帮助读者做好准备，以预判和应对新兴威胁。

本书在 Web 安全方面采取了均衡的方法，将扎实的理论基础与实践导向相结合。它提供了全面的视角，超越了纯理论讨论，融入了真实场景和实际渗透测试中的洞见，并通过详细的案例研究和实用示例，弥合了理论与实践之间的鸿沟。

由于本书面向更广泛的读者群体——从 Web 安全初学者到希望更新技能的经验丰富的专业人士，以及介于两者之间的所有人——并且涵盖了从基本概念到高级技术的广泛主题，它不仅会成为一份宝贵的资源，更将成为该领域从业者的一站式参考指南。

当你越读越深入时，你将被邀请去探索 Web 安全这个复杂而至关重要的世界。我希望在这段旅程结束时，你不仅能获得丰富的知识，还能掌握实用技能和战略思维，从而帮助你做好准备，应对并把握数字世界中的挑战与机遇。
