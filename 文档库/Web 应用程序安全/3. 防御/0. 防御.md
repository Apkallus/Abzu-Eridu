# 第三部分：防御

这是《Web 应用程序安全》的最后一部分。在第一部分和第二部分的基础上，我们将深入分析构建一个现代全栈 Web 应用所涉及的内容。在分析的每个环节，我们都会考虑重大的安全风险和关切。在提出关切之后，我们将评估替代实现方案以及能够缓解安全风险的应对措施。

在整个过程中，您将学习可以集成到软件开发生命周期中的技术，以减少生产代码中的漏洞。这些技术涵盖范围广泛，从默认安全的应用程序架构，到避免不安全的反模式，再到适当的安全导向代码审查技术，以及针对特定类型攻击的应对措施。

到第三部分结束时，您将在 Web 应用程序侦察、进攻性渗透测试技术和安全软件开发方面打下坚实的基础。届时，我鼓励您重新阅读前两部分中感兴趣的内容（但带着新的上下文），或者继续将您的新技能应用到现实世界中。

让我们继续前进，开始学习软件安全以及构建能够抵御黑客攻击的 Web 应用所需的技能。
