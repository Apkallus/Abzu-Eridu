# INTRODUCTION

想象在不久的未来，一名威胁行为者决定攻击一家跨国公司的数字资产，目标是埋藏在数百万美元基础设施背后、价值数亿美元的知识产权。理所当然地，攻击者首先启动了最新版本的 Metasploit。

在探测了目标的网络边界后，他们发现了一个薄弱点，并开始一系列有条不紊的攻击，几乎破坏了网络的每一个环节。然后，他们在系统中横向移动，识别出维持公司运营的关键业务组件。只需一次按键，他们就能危及公司所有的敏感数据。

奇怪的是，今天的渗透测试人员常常发现自己扮演着如上所述的虚构对手的角色，应那些需要高安全级别的公司的请求，执行合法的攻击。欢迎来到渗透测试的世界和安全的未来。

## 为什么要进行渗透测试？

公司投入数百万美元用于安全项目，以保护关键基础设施，识别防御中的薄弱环节，并防止严重的数据泄露。渗透测试是识别这些项目中系统性弱点和缺陷的最有效方法之一。通过尝试绕过安全控制和规避安全机制，渗透测试人员能够找出黑客可能破坏组织安全并损害整个组织的途径。

在阅读本书时，请记住，你的目标不一定只是针对一个或多个系统。你的目标是，以安全且可控的方式，展示攻击者如何可能对组织造成严重伤害，并影响其创收、维护声誉和保护客户等能力。

## 为什么选择 Metasploit？

Metasploot 不仅仅是一个工具；它是一个完整的框架，提供了自动化平凡、常规和复杂任务所需的基础设施。这种自动化使你能够专注于渗透测试中的专业领域，识别信息安全管理程序中的缺陷。

H.D. Moore 最初开发 Metasploit 时，他意识到自己在安全工作中大部分时间都花在验证和清理公开的漏洞利用代码上。他开始着手开发一个灵活的漏洞利用创建与开发框架，并于 2003 年 10 月发布了第一个基于 Perl 的 Metasploit 版本，共包含 11 个漏洞利用模块。后来，Metasploit 团队将该框架从 Perl 迁移到 Ruby，在安全社区中得到了广泛采用，用户贡献也大幅增加。随后，漏洞扫描领域的领导者 Rapid7 收购了它。

在首次发布二十年后，Metasploit 拥有超过 2400 个漏洞利用模块，以及数千个用于渗透测试前、后阶段任务的其他模块。随着你阅读本书各章节并建立起全面的方法论，你将开始理解在渗透测试中可以使用 Metasploit 的多种方式。Metasploit 允许你轻松构建攻击向量，以增强其漏洞利用、载荷、编码器等，从而创建和执行高级攻击。在本书的不同部分，我们将介绍一些第三方工具（包括本书作者编写的部分工具），它们基于 Metasploit 框架构建。到本书结束时，你应该能熟练使用该框架，理解高级攻击技术，并能够负责任地应用这些技术。让乐趣与挑战开始吧。

## 关于本书

本书旨在教你 Metasploit 框架的基础知识以及漏洞利用方面的高级技术。我们的目标是为初学者提供有用的教程，并为从业者提供参考。然而，我们不会一直手把手教你。编程知识在渗透测试领域是一个明显的优势，本书中的许多示例将使用 Ruby 或 Python 编程语言。尽管如此，虽然我们建议你学习一门编程语言以帮助进行高级漏洞利用和攻击定制，但编程知识并不是必需的。

随着你对 Metasploit 越来越熟悉，你会注意到该框架会频繁更新，添加新功能、漏洞利用和攻击手段。我们在编写本书时知道 Metasploit 在不断变化，并且没有一本纸质书能跟上这种快速开发节奏。因此，我们专注于基础知识，因为一旦你理解了 Metasploit 的工作原理，你就能快速适应其更新。

## 本版新增内容

我们已全面更新本版，以反映在 21 世纪第三个十年中使用 Metasploit 的经验。你将看到新模块、攻击技术和攻击面的覆盖，包括：

云渗透 – 本版新增一章关于渗透云环境的内容，包括权限提升技术和 Docker 容器逃逸。  
规避技术 – 我们介绍了创建可以规避防病毒系统二进制文件的策略，包括为 MSFvenom 反向 shell 创建自定义模板，以及使用 Metasploit 框架内置的规避模块。  
恶意文档生成 – 本版涵盖使用 Metasploit 生成恶意 Word 和 PDF 文档，专为客户端侧攻击定制。  
社会工程学与钓鱼 – 我们讨论了在钓鱼攻击中使用 Evilginx 等工具绕过某些双因素认证方法。此外，还讨论了部署 USB HID 设备（如 Rubber Ducky 和 O.MG 线缆）以传递载荷。  
基于 Wi-Fi 的攻击 – 我们增加了关于使用 Wi-Fi 工具（如 Wi-Fi Pineapple 执行邪恶双子攻击，以及使用 Alfa 网卡监控和破解 Wi-Fi 网络、中断客户端连接）的内容。我们还讨论了如何向移动设备发送恶意 APK 文件。  
Active Directory 攻击 – 我们探讨了 DCSync 和黄金票据攻击等技术。
