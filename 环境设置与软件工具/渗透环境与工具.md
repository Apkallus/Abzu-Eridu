## 集成工具
### Burp
#### 安装
1. [Burp 社区版下载](https://portswigger.net/burp/communitydownload)
2. 在浏览器中手动设置代理为 Burp 的监听地址
3. [安装Burp CA 证书](https://portswigger.net/burp/documentation/desktop/external-browser-config/certificate)：
    1. 使用浏览器访问拦截代理服务器的地址下载证书
    2. 在浏览器设置中的证书管理器中导入此证书，确保勾选此证书可以标识网站

#### 设置
- 使用暗色 UI：设置 - User interface - Display - Appearance - Theme 选择 Dark
- 更改默认信息显示字体为 Consolas
- 开启配置缩放设置
    - 光标偏移错位修复
    <https://www.52pojie.cn/thread-1908732-1-1.html>
- 使用 Burp 内置 Chrome 浏览器或在 Firefox 的代理设置中添加排除项 ```firefox.com, mozilla.com, mozilla.net, mozilla.org, darkreader.org, bing.com, googleapis.com, linkedin.com, youtube.com, portswigger.net, google.com, immersivetranslate.com, immersivetranslate.cn``` 或在 burp 中设置范围

#### 使用
- [官网文档](https://portswigger.net/burp/documentation/desktop)
    部分文档拥有视频操作与对应的 lab 实践，在主题 lab 的介绍与 lab 的示例解决方案中包括逐步的使用方式。

- 插件
    :warning: 若出现功能异常，检查扩展设置或关闭扩展后重试
