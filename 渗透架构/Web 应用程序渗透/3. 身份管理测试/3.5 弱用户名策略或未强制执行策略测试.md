# 弱用户名策略或未强制执行策略测试

## 概述

用户账户名通常具有高度结构性（例如：Joe Bloggs 的账户名为 jbloggs，Fred Nurks 的账户名为 fnurks），有效的账户名很容易被猜测。

## 测试目标

- 确定一致的账户名结构是否会导致应用程序容易受到账户枚举攻击。
- 确定应用程序的错误信息是否允许进行账户枚举。

## 测试方法

- 确定账户名的结构。
- 评估应用程序对有效和无效账户名的响应。
- 利用对有效和无效账户名的不同响应来枚举有效账户名。
- 使用账户名字典来枚举有效账户名。

## 修复措施

确保在登录过程中输入无效账户名、密码或其他用户凭证时，应用程序返回一致的通用错误信息。
