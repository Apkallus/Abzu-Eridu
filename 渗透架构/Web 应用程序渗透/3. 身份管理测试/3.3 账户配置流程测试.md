# 账户配置流程测试

## 摘要

账户配置流程可能被攻击者利用，使其能够在未经过正当身份验证和授权流程的情况下创建有效账户。

## 测试目标

- 验证哪些账户可以配置其他账户及其账户类型。

## 测试方法

确定哪些角色能够配置用户账户以及可以配置的账户类型。

- 对账户配置请求是否有任何验证、审核和授权流程？
- 对账户取消配置请求是否有任何验证、审核和授权流程？
- 管理员能否配置其他管理员账户，还是只能配置普通用户账户？
- 管理员或其他用户能否配置权限高于自身权限的账户？
- 管理员或用户能否取消配置自己的账户？
- 被取消配置用户所拥有的文件或资源如何处理？是否会被删除？访问权限是否会转移？
