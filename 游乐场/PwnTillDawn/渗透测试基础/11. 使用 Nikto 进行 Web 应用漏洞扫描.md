# 使用 Nikto 进行 Web 应用漏洞扫描

在测试 Web 应用程序时，手动查找各种安全问题可能是一项艰巨的任务。这是因为 Web 服务器和 Web 应用在所使用的技术和配置上都具有极高的多样性。

因此，渗透测试人员经常使用像 Nikto 这样的自动化 Web 扫描器，在他们寻找隐藏的安全缺陷时，发现一些容易获取的目标。

Nikto 是一个方便的开源（GPL）Web 服务器和应用程序扫描器，它可以扫描数千种潜在的漏洞和弱点，同时也能发现一些有趣的东西，为你提供初步的立足点。它试图在尽可能短的时间内测试 Web 服务器，因此，它的活动在日志文件中非常明显，并且会被入侵防御系统（IPS）和入侵检测系统（IDS）检测到。

另一个重要说明是，并非每个检测到的问题都一定是正确的，需要手动验证来确认 Nikto 的发现。

有关其功能的完整列表，请参阅 <https://cirt.net/Nikto2> 网页。

## 如何使用 Nikto 进行扫描？

使用 Nikto 启动扫描非常简单：

```bash
nikto -output nikto-scan.txt -host http://10.150.150.11/
```

`-output` 参数指定结果保存的文件，`-host` 用于设置扫描目标。
