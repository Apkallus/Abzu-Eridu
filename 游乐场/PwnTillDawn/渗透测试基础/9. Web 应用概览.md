# Web 应用概览

Web 应用无处不在，并且在大多数情况下是公司的核心组成部分。它们被用于多种目的，并且经常捕获、处理、存储和传输敏感数据（机密商业信息、员工信息、客户个人数据、财务信息等）。

事实如此：Web 应用是网络犯罪分子获取敏感信息并入侵组织的最简单途径之一。众多研究证实了这一点，这也是我们为什么会有多个课程涵盖 Web 应用的原因。

Web 应用（或 Web 应用程序）是一种运行在 Web 服务器上的应用软件。用户通过具有活跃互联网连接或局域网的 Web 浏览器访问 Web 应用。这些应用程序使用客户端-服务器模型进行编程。

Web 应用可以用多种不同的语言（PHP、ASP、Python 等）开发，用于信息检索和处理。在客户端，我们使用 JavaScript、HTML 等技术来向用户呈现信息。

## Web 应用如何工作

有些应用是动态的，需要服务器端处理。其他则是完全静态的，不需要服务器处理。

Web 应用需要一个 Web 服务器来管理来自客户端的请求，一个应用服务器来执行请求的任务，以及有时需要一个数据库来存储信息。应用服务器技术范围包括 ASP.NET、ASP、ColdFusion、PHP、JSP 等等。

一个常见的 Web 应用工作流程如下：

1. 用户通过网络浏览器向 Web 服务器触发一个请求。
2. Web 服务器将该请求转发给相应的 Web 应用服务器。
3. Web 应用服务器执行请求的任务（例如查询数据库或处理数据），然后生成结果。
4. Web 应用服务器将结果发送回 Web 服务器。
5. Web 服务器将所请求的信息响应回客户端，然后该信息会显示在用户的屏幕上。

## 一些例子

我们可以提到一些 Web 应用的例子，例如在线商店、内容管理系统、文件管理器、表单、电子学习平台、流媒体平台等。
