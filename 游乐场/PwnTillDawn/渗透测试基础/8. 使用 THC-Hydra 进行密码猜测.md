# 使用 THC-Hydra 进行密码猜测

THC-Hydra 是一款针对网络服务执行密码猜测攻击的优秀工具。

Hydra 最大的优点之一是它支持对大量基于网络的服务进行密码猜测，例如：SSHv2、Telnet、FTP、HTTP、HTTPS、HTTP-PROXY、SMS、SMBNT、MS-SQL、MYSQL、REXEC、RSH、RLOGIN、CVS、SNMP、SMTP-AUTH、SOCKS5、VNC、POP3、IMAP、NNTP、PCNFS、ICQ、SAP/R3、LDAP2、LDAP3、Postgres、Teamspeak、Cisco auth、Cisco enable、Cisco AAA 等。

## 入门指南

一旦你确定了要暴力破解的服务，可以输入类似下面的命令：

```bash
hydra -l root -P /usr/share/wordlists/rockyou.txt.gz -t 6 ssh://10.150.150.11 -I
```

上述命令将对目标 10.150.150.11 上运行的 SSH 服务，针对用户 “root” 使用密码字典 “rockyou” 执行密码暴力破解攻击，并开启 6 个线程。

某些服务对暴力破解处理不佳，因此你可以使用参数 `-t 1` 或 `-t 4` 来减少并行连接数，并使用 `-W 1` 或 `-W 3` 在连接之间设置等待时间，以便让服务器恢复。
