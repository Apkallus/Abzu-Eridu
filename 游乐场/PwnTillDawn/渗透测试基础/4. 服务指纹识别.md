# 服务指纹识别

正如我们在第一课中已经看到的，对目标进行侦察时最重要的步骤之一包括扫描服务器端口以发现正在运行的服务。

在进行攻击时，知道端口21是开放的并不能给我们足够的信息；真正有帮助的是了解运行的是哪个FTP服务器及其版本，以便制定最佳的攻击策略。

## 我们如何发现正在运行的服务？

我们在上一课中已经介绍了nmap。nmap是一个非常强大的工具，在许多情况下，它可以让我们发现服务器上运行的服务的重要信息。

使用其service-probes数据库，nmap试图发现：

- 服务协议
- 应用程序名称
- 版本号
- 主机名
- 设备类型
- 操作系统

使用nmap的版本检测功能非常简单：

```bash
nmap -sV -p- hostname/IP
```

另一个选项是使用`-A`参数，它可以在其他一些有趣且有用的功能中启用版本检测。
