# 实验：通过兄弟域（sibling domain）绕过 SameSite Strict

本实验的实时聊天功能易受跨站 WebSocket 劫持（CSWSH）攻击。为了解决本实验，请登录受害者的账户。

为此，请使用提供的漏洞利用服务器执行 CSWSH 攻击，将受害者的聊天记录窃取到默认的 Burp Collaborator 服务器。聊天记录中包含明文登录凭据。

如果你尚未完成此操作，我们建议你在尝试本实验之前先完成关于 WebSocket 漏洞的主题学习。

> **提示**
> 确保你全面审计所有可用的攻击面。留意可能有助于你实施攻击的其他漏洞，并记住两个域可以位于同一个站点内。
