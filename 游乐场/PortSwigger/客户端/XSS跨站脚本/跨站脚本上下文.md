```javascript
  toString=x, /* 将匿名函数赋值到 window.toString 方法，无需显式引用window对象即可访问其属性和方法 */
  window+'',  /* 进行字符串操作，触发匿名函数 */
  {x:''}) /* 处理尾部 */
  .finally(_ => window.location = '/')
```
