# 内核安全

内核是操作系统的核心组件，充当软件与硬件之间的桥梁。内核运行在操作系统的最底层，其访问权限极其深刻，甚至可以视同于伪装成系统本身，超越了 root 用户的最高权限。内核安全至关重要，因为一旦这一层面被攻破，攻击者就能以系统的身份行事。漏洞可能导致未经授权的数据访问、系统崩溃或 rootkit 的悄无声息地安装等后果。

为了开始内核挑战，你需要在虚拟机内运行这些挑战。你可以通过在终端中运行命令 `vm start` 来启动此虚拟机。虚拟机将自动加载位于 `/challenge` 目录中的内核模块。然后，你可以使用 `vm connect` 将终端连接到该虚拟机！
