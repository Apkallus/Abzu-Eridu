# 返回导向编程

想象自己是一位数字指挥大师，在广袤的数字领域中指挥着一曲代码的交响乐。
然而，转折来了：你无法亲自谱写自己的音符。
相反，你得到的是一份散落在系统中的现有代码片段遗产。
这正是返回导向编程（ROP）漏洞利用的精髓！仅凭系统自身代码的残留片段，你便能编织出一首灵巧的乐章，使其按你的旋律起舞，优雅而隐蔽地绕过现代安全防护措施。

每个代码片段都像一个乐句，以一条“返回”指令结尾，将你快速带到隐秘协奏曲中的下一个片段。
随着每一次跳转，你编织出一个邪恶的故事，绕过安全检查，执行未经授权的操作，而这一切都在系统防御机制的不知不觉中进行。

ROP 不仅仅是一种黑客技术；它是一件未经授权的编排杰作，一场借来指令的芭蕾舞，经过精准的编舞，以实现你隐秘的目标。
借助 ROP，你踏入一个领域，其中每个字节都是一个节拍，每个返回都是一个节奏，开启一段激动人心的漏洞利用与发现之旅。
