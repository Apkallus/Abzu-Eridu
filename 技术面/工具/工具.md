## 密码

### Hydra

`Syntax: hydra [[[-l LOGIN|-L FILE] [-p PASS|-P FILE]] | [-C FILE]] [-e nsr] [-o FILE] [-t TASKS] [-M FILE [-T TASKS]] [-w TIME] [-W TIME] [-f] [-s PORT] [-x MIN:MAX:CHARSET] [-c TIME] [-ISOuvVd46] [-m MODULE_OPT] [service://server[:PORT][/OPT]]`

- `-l 目标用户名 | -L 目标用户名文件`
- `-p 密码 | -P 密码字典路径`
- `-t 线程数`
- `协议://目标IP地址`

### ffuf - 模糊测试

### Nikto - 扫描器

- 程序路径 `/nikto/program/nikto.pl`
- `-output 导出文件` 
- `-host URL地址`

- `-Tuning -扫描调优选项` 设置扫描类型

    ```log
    1     有趣的文件 / 在日志中出现过
    2     错误配置 / 默认文件
    3     信息泄露
    4     注入（XSS/脚本/HTML）
    5     远程文件获取 - Web 根目录内
    6     拒绝服务
    7     远程文件获取 - 服务器范围
    8     命令执行 / 远程 Shell
    9     SQL 注入
    0     文件上传
    a     认证绕过
    b     软件识别
    c     远程源包含
    d     Web 服务
    e     管理控制台
    x     反向调优选项（即包含除指定项外的所有）
    ```

### Burp

安装
- 下载后在浏览器中手动设置代理为 Burp 的监听地址
- [安装Burp CA 证书](https://portswigger.net/burp/documentation/desktop/external-browser-config/certificate)：
    1. 使用浏览器访问拦截代理服务器的地址下载证书
    2. 在浏览器设置中的证书管理器中导入此证书，确保勾选此证书可以标识网站

设置
- 用户界面选项 `设置 - User interface`
    - Display
        - 主题选择 Dark 暗黑模式
        - 界面放大：设置缩放为默认的固定 `1.0` ，尝试调大字体来控制字体清晰度以及界面缩放。若仅调整缩放则光标错位。
            - 字体20
            - 开启配置缩放
                - 光标偏移错位修复
                <https://www.52pojie.cn/thread-1908732-1-1.html>
    - Message editor
        - 更改默认信息显示字体为 Consolas
- 内置 chrome 浏览器
    - appearance 
        - theme 主题设置 dark
        - 自定义字体大小 18
        - 页面缩放 125%


使用
- [官网文档](https://portswigger.net/burp/documentation/desktop)
    部分文档拥有视频操作与对应的 lab 实践，在主题 lab 的介绍与 lab 的示例解决方案中包括逐步的使用方式。

- 插件
    :warning: 若出现功能异常，检查扩展设置或关闭扩展后重试

### Zap

安装
1. [Zap 下载](https://www.zaproxy.org/download/)

设置
- 工具 - 选项 - 显示 - 外观
    - 选择字体“微软雅黑”，设置字体大小16
    - 使用暗色 UI：选择 Flat Dark
    - `显示-最大化左边的标签` 或 `最大化底部标签` 同时显示历史记录与请求
    - `响应面板位置-面板并排` 同时显示请求与响应

使用
- [官网文档](https://www.zaproxy.org/docs)
    - 拥有 Burp 到 Zap 的[功能映射专题](https://www.zaproxy.org/docs/burp-to-zap-feature-map/)
    - 内置浏览器扩展的[使用](https://www.zaproxy.org/blog/2021-11-26-launching-browsers-with-extensions/)
        每次启动内置浏览器将使用全新的配置文件，使用扩展需在 `Selenium` 选项中手动添加浏览器扩展组件。
        对于 Firefox，在安装附加组件页面的蓝色按钮即为扩展的文件地址，保存到本地后添加到 `Selenium` 的浏览器扩展功能中
- :warning: 插件“沉浸式翻译”的划词翻译功能在内置浏览器中不可用，而在 Zap 为代理的浏览器中可用
