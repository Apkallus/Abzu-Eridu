后渗透利用阶段在你攻陷一个或多个系统后开始。

后渗透利用针对特定系统，识别关键基础设施，并关注公司最重视并试图保护的信息或数据。

在后渗透利用中攻击系统时，你应该花时间确定各个系统的功能以及它们不同的用户角色。

攻破了面向互联网的主机
- 检查它是否连接到内部网络
    - ipconfig
- 内网探测
    - ARP 探测
    - SOCKS 与代理链进行 Nmap 扫描
        - nmap -> proxychains -> msf socks_proxy -> msf meterpreter sessions

入侵了一个基于 Windows 的系统。我们可以使用 Meterpreter 控制台从该系统中提取哈希值，然后使用这些凭据对其他基于 Windows 的系统进行身份验证。本地管理员帐户几乎总是从一个系统到另一个系统都相同，因此即使在企业环境中，我们也可以使用来自一个系统的信息将攻击桥接到另一个系统。
