## 终端

`Ctrl-R` 搜索历史命令，实时匹配当前输入
- `回车` 选定并执行
- `→` 右方向键选定并边界 

`.bashrc` 文件
- `PS4 & set` 格式显示指令
- `ls` 列出挑战目录内容
- `rm/mkdir/cp` 复制挑战目录内容到当前工作目录
- `mapfile/file/grep/sed` 将选定类型文件保存为数组批量处理
    - `basename` 从路径提取文件名
        - 使用 `binary.ninja` 静态分析
    - `checksec` 查看 elf 文件安全信息
- `ropper/ROPgadget` 查找 ROP 工具
    - `ropper` 对内核文件查找时在挑战中 Killed
- `objdump/grep` 查看 ret2cus 地址
- `/proc/cpuinfo` 查看 cpu 支持 
    - `shstk|ibt`

`dmesg -W &` 持续输出最新内核日志
`ps aux | grep` 打印进程信息
`kill -s 9` 终止进程
`while true; do 命令; sleep 0.5; done` 持续执行指令

结构：
- `pahole -C _IO_FILE /lib/x86_64-linux-gnu/libc.so.6` 查看数据结构布局，以确认字段偏移
- gdb 对指针进行强制类型转换后查看结构或字段
- `grep -R "struct _IO_FILE" /usr/include 2>/dev/null` 头文件中查找
- elixir.bootlin.com 查找
- 在编译的头 include 路径中查找常量/结构/原型
    `grep -r SOCK_STREAM /usr/include/x86_64-linux-gnu`

gdb:
- 在程序添加 `input()` 设置暂停
- gdb 运行到断点等待输入
- 使用 `Ctrl-B 方向键` 切换窗口到程序
- 按任意键继续运行,触发 gdb 断点

rop/shellcode:
- 使用 pwntool 的 `ELF()` 得到程序对象
    - `elf.address` 设置基址
    - `ROP(elf)` 设置 rop 对象
- 使用 `ROPgadget --binary` 或 `rp++ --unique -r2 -f` 或 `ropper --file` 查看工具
    - 在 pwntool 的 rop 对象使用 `rop.find_gadget(['pop rax', 'ret']).address` 得到工具地址
- `ldd` 查看链接的库文件路径

逆向：
- 列出函数参数表
- 列出结构字段与偏移
- 对应参数与结构字段
