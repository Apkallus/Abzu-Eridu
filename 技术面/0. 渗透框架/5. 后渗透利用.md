## Microsoft NetBIOS、SMB 和 RPC 协议循环

流程
- 从三种协议暴露接口中收集各种信息：端点、用户名
- 使用得到的用户名列表攻击暴露的认证机制
    - SMB，使用 Hydra
        - NetBIOS 会话服务
        - SMB Direct 服务
    - WMI，使用 WMICracker 或 ebrute
        - RPC 定位器服务

    本地 Administrator 账户默认不锁定，使其成为暴力破解的有吸引力的目标。

    Windows 域通常会强制执行账户锁定策略，因此只有在了解策略（或缺乏策略）后，才应考虑激进的密码猜测。否则可能导致整个域被锁定！一个明智的替代方案是使用少量已知密码对许多有效账户和服务接口进行横向暴力攻击。

## 身份验证和使用访问权限

在使用 SMB 和 Microsoft RPC 端点进行身份验证后，从系统获取材料、提升权限并转向访问其他应用程序和服务。步骤：

- 使用 SMB 进行身份验证
    - 账户密码登录
        - Windows `net`
        - Unix `smbclient`
    - NTLM 哈希传递 `Mimikatz sekurlsa::pth`
    - Kerberos 票据
- 查询 WMI 以了解系统配置
    - 工具
        - WMIdump
        - Impacket 库
           
    - 获取信息
        - 操作系统配置
        - 本地用户账户和组
        - 正在运行的进程、服务和配置
        - 已安装的软件包、服务包和热修复
- 远程命令执行

    Impacket 库：
    | 脚本 | 接口 | 说明 |
    |------|------|------|
    | `smbexec.py` | `\pipe\svcctl` via SMB | 将命令 shell 作为服务上传和执行 |
    | `psexec.py` | | |
    | `services.py` | | 启动和停止任意系统服务（例如终端服务或任务计划程序） |
    | `atexec.py` | `\pipe\atsvc` via SMB | 通过任务计划程序执行命令 |
    | `wmiexec.py` | DCOM via 端口 135 | 在不触碰磁盘或不运行新服务的情况下隐蔽地执行命令 shell |

    这些实用程序支持通过密码、NTLM 哈希和 Kerberos 票据进行身份验证。

    防病毒软件通常会对通过 `smbexec.py` 和 `psexec.py` 上传的恶意内容发出警报并将其删除。因此，建议使用 `wmiexec.py` 以及 Metasploit `web_delivery` 模块来生成功能丰富的 shell，且不会被标记（因为没有任何内容写入磁盘）

- 访问和修改注册表
    - 工具：Microsoft `regdmp.exe`、`regini.exe` 和 `reg.exe` 实用程序
    - 功能：安装后门
- 获取秘密（密码、哈希、长期密钥和票据）
    - 各种秘密包括：
        - 内存中的明文凭据（即账户密码）
        - 可被破解或传递的长期密钥和 NTLM 哈希
        - Kerberos 票据授予票据和各个服务票据
        - 客户端软件中存储的凭据（例如浏览器和邮件客户端）
        - 浏览器中的自动完成字段

    - 工具
        - mimikittenz
        - Impacket 的 `secretsdump.py`
        - NirSoft 的密码恢复工具

    - 其他方式
        - 访问磁盘上的注册表配置单元，可使用 `creddump` 实用程序提取 NTLM 哈希、缓存的域密码和 LSA 秘密。


自动化过程
- `Auto Domain Admin and Network Exploitation（autoDANE）` 实用程序[^19]

## Microsoft 服务测试

在测试 Microsoft 服务时，应遵循以下途径：

- 扫描以枚举通过 TCP、UDP、SMB 和 HTTP 暴露的服务。
- 查询 SMB 和 RPC 服务以枚举系统配置。
- 调查弱点（例如 SMB 和 RPC 实现中的缺陷），这些缺陷可以在没有凭据的情况下远程利用，导致用户枚举、代码执行或特权访问。
- 对暴露的 SMB、WMI 和 RDP 接口进行暴力密码猜测。
- 在身份验证后进一步查询、执行命令、收集凭据和提升权限。
