二进制工具
- ldd 显示程序在运行时加载的共享库
- Objdump 命令行反汇编
	- -j 字段。显示特定字段，如.plt.text
- strings 查看文件字符串
	- -t 基数，x 十六进制。-tx 同时显示十六进制偏移量
- strace 追踪系统调用与信号
	- -e trace=syscall。查看特定系统调用
	- -e inject=syscall:delay_enter=微秒时间 设置调用前延迟。delay_exit 对应调用后延迟
- ltrace 追踪共享库与系统调用
	- -S显示系统调用
- checksec 解析elf头部的编译缓解设置
- libc-database 基于符号偏移量识别libc版本
	- get 版本。获取所有或指定libc版本文件
	- find 符号 偏移量。识别libc版本
- libc.blukat.me 在线查看 libc database
- patchelf 修改elf文件库，以轻松切换libc版本来分析
- readelf 
	- -s 获取符号偏移量
- one_gadget 在libc中查找单个工具获取shell
	- 手动
		- strings 获取 shell 路径字符串偏移量
		- Objdump 查找对目标字符串的引用
	- one_gadget 工具，查找工具并显示约束
- ropper 生成rop链或搜索代码片段工具
	- 拥有根据约束条件搜索工具链的功能。如 mprotect address=1337 size=0xcaffe
	- 拥有语义搜索功能。如，栈指针增加、避免特定寄存器破坏
