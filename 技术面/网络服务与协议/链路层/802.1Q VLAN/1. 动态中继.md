在加固的环境中，你的端口将具有静态分配，将你限制在特定的 VLAN 中。然而，许多交换机默认支持**动态中继协议（DTP）**，攻击者可以滥用该协议来模拟交换机并接收所有 VLAN 上的流量。

Cisco 交换机使用的 802.1Q 端口模式

    | 模式 | 描述 |
    |------|------|
    | Access | 将端口置于永久非中继模式 |
    | Trunk | 将端口置于永久中继模式 |
    | Dynamic auto | 如果相邻端口协商中继或 dynamic desirable 连接，端口可能将链路转换为中继（默认模式） |
    | Dynamic desirable | 端口主动尝试将其链路转换为中继，如果相邻端口协商中继、dynamic desirable 或 dynamic auto 模式，则成为中继端口 |
    | Nonegotiate | 完全禁用端口的动态中继 |
