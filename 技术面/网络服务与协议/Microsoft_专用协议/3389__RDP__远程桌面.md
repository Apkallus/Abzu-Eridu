# 远程桌面服务

概念：
- Windows 终端服务器驱动程序（`termdd.sys`）通过 TCP 端口 3389 上的 RDP 提供访问。包括远程桌面连接和远程桌面共享在内的客户端可以访问桌面和特定应用程序。

攻击面：
- 暴力密码猜测
    - Hydra
    - Ncrack
- 中间人攻击
    - 使用 Nmap `rdp-enum-encryption` 测试服务器是否支持弱密码协议
- 通过软件缺陷

其他工具：
- rdp-sec-check
