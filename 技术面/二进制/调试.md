
使用 root 权限调试

缩小范围：
- 将代码分割为代码块
- 将代码块设置为局部函数
- 代码块/函数前后打印信息
- 设置断点到函数


检查内容：
- 函数参数
- 分支条件
- 查看反汇编代码
- 类型结构
- 常量


打印信息：
- 添加调试的全局变量 flag
- 爆破进度
- 除普通程序外，也在 shellcode 中打印调试信息
- dmesg
    - 当进行覆盖操作后报错，错误信息的寄存器是否使用了相邻的值，可能清空后被使用

定位：
- python input 停止在选定位置，gdb attach 连接以调试特定位置
- asm shellcode 添加 int3 设置固定断点

同步：
- 打印的提示语
- 操作间的 sleep
- 内存/指令的 fence
- fork 后父进程的 os.wait

gdb:
- 使用函数设置状态

断言：
- 遇到非预期值，立即崩溃以减少调试或尝试的时间
    - 异常仅为地址随机导致：字符串型写入读取原语中，在链表劫持代码中加入 safe-linking 混淆后的空字节或空格等排除字符的断言
