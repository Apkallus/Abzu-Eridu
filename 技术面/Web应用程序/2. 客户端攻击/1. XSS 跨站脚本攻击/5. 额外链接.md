## 额外进阶

- **Sammyworm 分析**：深入研究 "Sammy worm" 事件及其对社交网站 MySpace.com 的影响。调查攻击者的方法，了解其执行方式，并探索潜在的复现策略。
- **用于 XSS 预防的 Trusted Types**：研究 Trusted Types 的实施及其在缓解 DOM XSS 攻击中的作用。重点识别可能导致绕过的边缘情况和常见错误配置。
- **DOM Goat 中的 DOM XSS 练习**：通过完成 DOM Goat [https://domgo.at/] 中所有可用的 DOM XSS 练习，进行实践学习。
- **Masato 的 mXSS 案例研究**：检查 Masato 的 mXSS 案例，该案例导致了 Google.com 上的 XSS 漏洞。探索此漏洞的根本原因，并了解如何将类似的有效载荷用于针对其他清理器和净化器 [www.youtube.com/watch?v=lG7U3fuNw3A]。
- **沙盒绕过技术**：查阅 PortSwigger Academy 上关于沙盒绕过的全面参考资料。该资源为公开已知的绕过方法提供了宝贵的见解 [https://portswigger-labs.net/angular_dom_based_sandbox_escapes/]。
- **高级 DOM 劫持方法**：研究 DOM 劫持中的复杂技术，重点关注多级劫持以及如何利用它来实现 DOM XSS [https://book.hacktricks.xyz/pentesting-web/xss-cross-site-scripting/dom-clobbering]。
