### Hydra

`Syntax: hydra [[[-l LOGIN|-L FILE] [-p PASS|-P FILE]] | [-C FILE]] [-e nsr] [-o FILE] [-t TASKS] [-M FILE [-T TASKS]] [-w TIME] [-W TIME] [-f] [-s PORT] [-x MIN:MAX:CHARSET] [-c TIME] [-ISOuvVd46] [-m MODULE_OPT] [service://server[:PORT][/OPT]]`

- `-l 目标用户名 | -L 目标用户名文件`
- `-p 密码 | -P 密码字典路径`
- `-t 线程数`
- `协议://目标IP地址`

### ffuf - 模糊测试

- 用法：在爆破位置设置 FUZZ 关键字
- `-w 字典路径`

### Nikto - 扫描器

- 程序路径 `/nikto/program/nikto.pl`
- `-output 导出文件` 
- `-host URL地址`

- `-Tuning -扫描调优选项` 设置扫描类型

    ```log
    1     有趣的文件 / 在日志中出现过
    2     错误配置 / 默认文件
    3     信息泄露
    4     注入（XSS/脚本/HTML）
    5     远程文件获取 - Web 根目录内
    6     拒绝服务
    7     远程文件获取 - 服务器范围
    8     命令执行 / 远程 Shell
    9     SQL 注入
    0     文件上传
    a     认证绕过
    b     软件识别
    c     远程源包含
    d     Web 服务
    e     管理控制台
    x     反向调优选项（即包含除指定项外的所有）
    ```



### Zap

安装
1. [Zap 下载](https://www.zaproxy.org/download/)

设置
- 工具 - 选项 - 显示 - 外观
    - 选择字体“微软雅黑”，设置字体大小16
    - 使用暗色 UI：选择 Flat Dark
    - `显示-最大化左边的标签` 或 `最大化底部标签` 同时显示历史记录与请求
    - `响应面板位置-面板并排` 同时显示请求与响应

使用
- [官网文档](https://www.zaproxy.org/docs)
    - 拥有 Burp 到 Zap 的[功能映射专题](https://www.zaproxy.org/docs/burp-to-zap-feature-map/)
    - 内置浏览器扩展的[使用](https://www.zaproxy.org/blog/2021-11-26-launching-browsers-with-extensions/)
        每次启动内置浏览器将使用全新的配置文件，使用扩展需在 `Selenium` 选项中手动添加浏览器扩展组件。
        对于 Firefox，在安装附加组件页面的蓝色按钮即为扩展的文件地址，保存到本地后添加到 `Selenium` 的浏览器扩展功能中
- :warning: 插件“沉浸式翻译”的划词翻译功能在内置浏览器中不可用，而在 Zap 为代理的浏览器中可用

### dnsdict6 - ipv 地址枚举

https://github.com/vanhauser-thc/thc-ipv6

`./dnsdict6 [-d4] [-s|-m|-l|-x|-u] [-t THREADS] [-D] domain [dictionary-file]`

`-t` 线程数，默认8
`-[smlxu]` 字典大小
    -s(mall=100), -m(edium=1419) (DEFAULT)
           -l(arge=2601), -x(treme=5886) or -u(ber=16724)
