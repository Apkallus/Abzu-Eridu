使用署名共享、版权声明许可类材料构建框架

- 使用 LLM 翻译到中文并修改
    - 部分冗长内容可精简为一小段话
    - 部分对原理及向量的描述是否可使用链接到对应网站？
    - 当前使用链接的原理及向量是否应当直接合并到正文？

- 其中，若包含不发布解决方案的声明，则仅对靶机的概念进行分析。

对于几乎禁止公开的站点，不直接使用翻译后的材料，而是记录对材料及靶机的概念分析。

综合各种文档与靶机扩充知识框架。

完成靶机后，回顾并分析记录漏洞根源及其对应的 CWE ID 与进行靶机本身同样重要。

对材料分类：
- 全文分析，信息密度高的博客文章或教材，花费足量时间。
- 仅查看对应内容，大型文档中特定信息相关且立即可应用，分析内容后转为速查手册。
- 速查手册，对于包含信息但缺乏结构或立即应用的材料。

内容记录方式：
对于基础知识，仅记录实际操作的细节与简略概念。  
对于渗透知识，记录所有，除非目标文档结构与内容良好而可直接翻阅
